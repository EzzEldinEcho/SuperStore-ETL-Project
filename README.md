# SuperStore ETL Project

## Project Overview
This project demonstrates a complete end-to-end Data Engineering pipeline using the Superstore dataset. It covers data exploration, cleaning, data warehousing, and reporting.

## Project Structure

- **Dataset**: Contains the raw and processed data files.
- **Cleaning_Stage**: Jupyter notebooks for Exploratory Data Analysis (EDA) and data cleaning.
- **DWH**: Data Warehouse schema and SQL scripts (SQL Server).
- **Superstore**: SSIS (SQL Server Integration Services) project for the ETL process.
- **Reports**: Power BI dashboard (`Superstore_Analysis_DWH.pbix`) for visualizing insights.
- **Docs**: Project documentation.

## Goals
1.  **Ingest**: Extract data from CSV files.
2.  **Transform**: Clean and transform data using Python and SSIS.
3.  **Load**: Load data into a Star Schema Data Warehouse.
4.  **Visualize**: Create interactive dashboards to analyze sales, profits, and trends.
